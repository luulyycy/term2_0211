# term2_0211
- 怎麼辦！我的行李 - 智慧行李打包分析程式介紹
# 🌟 程式概述
- 這是一個全方位的智慧行李打包分析工具，專為解決旅行者「不知道該帶什麼」、「帶太多或帶太少」的煩惱而設計。使用者只需輸入目的地、出發日期、過夜天數、性別以及是否需要泳衣，系統就會自動運用完整的數學公式，計算出精準的行李清單，並提供專業的行李箱尺寸建議。

# 🎯 核心功能特色
## 1. 智慧天氣分析
- 系統會根據您輸入的目的地與出發日期，自動查詢該地區當月份的氣候資料（包含溫度範圍、天氣狀況、季節），並據此調整衣物建議：

冬季（❄️）：外套建議攜帶2件，衣物類型以保暖為主

夏季（☀️）：外套建議攜帶1件，衣物類型以輕薄為主

若查無當月資料，系統會以季節判斷作為參考

## 2. 完整數學公式計算
- 所有物品數量皆由精準公式自動算出，完全不用手動計算：

- 物品類別	計算公式
上衣	天數 + 2
褲子/裙子	天數 ÷ 2（無條件進位）
內衣褲	天數 + 3
襪子	天數 + 2
睡衣	天數 ÷ 3（無條件進位）
外套	冬季2件 / 夏季1件
口罩	天數 + 5
收納袋	天數 ÷ 2（無條件進位）
化妝棉	天數 × 5
個人藥物	天數份
現金	預算 × 0.3（提示）
## 3. 八大類物品完整收錄
- 👔 衣物類
上衣、褲子/裙子、內衣褲、襪子、睡衣、外套、帽子、墨鏡、泳衣

- 女性額外包含：胸罩（與內衣褲同步）、衛生棉（✓標記）

- 🧴 盥洗用品
牙刷、牙膏（50g）、洗髮精（80ml）、沐浴乳（80ml）、洗面乳（50ml）

化妝棉、毛巾、刮鬍刀、刮鬍泡（50ml）、梳子

- 💄 美妝保養（男女有別）
女性：化妝水（50ml）、乳液（50ml）、防曬（50ml）、粉底（30ml）、口紅（2支）、眼影、睫毛膏、卸妝油（50ml）

男性：防曬（50ml）

- 📱 3C電子
手機、充電線、充電頭、行動電源、耳機、轉接頭

- 選帶：平板/筆電、相機、記憶卡

- 📁 證件錢包
護照、身分證、健保卡、信用卡（2張）、現金、飛機票、訂房紀錄（皆以✓標記）

- 💊 藥品
腸胃藥、止痛藥、感冒藥、創可貼（5片）、個人藥物（天數份）、蚊蟲藥

- 🎒 雜項
口罩（天數+5）、濕紙巾、衛生紙、水壺、雨傘

- 選帶：鎖頭、筆+筆記本

- 購物袋

- 🧳 行李箱
登機箱、背包、收納袋（天數÷2）、鞋袋

## 4. 行李箱尺寸智慧建議
- 根據您輸入的過夜天數，系統會自動推薦最適合的行李箱尺寸：

- 天數範圍	建議尺寸	適用情境
1～14天	24～26吋	短期長旅，最剛好
15～30天	28吋	一個月旅行，最實用
31～90天	29～30吋	1~3個月長期旅居
90天以上	32吋	3個月以上～半年、一年，最大托運行李箱
## 5. 剩餘空間個人化選擇
- 您可以選擇希望行李箱剩餘的空間百分比：

5%（幾乎裝滿）

10%（預留少許空間）

20%（寬鬆打包）

系統會以使用率進度條顯示，讓您清楚了解物品佔比。

## 6. 兩大分類一目了然
所有物品分為兩大類顯示：

- 👤 跟身類（隨身攜帶）
證件錢包、3C隨身、藥品隨身、雜項隨身

適合放在隨身背包、不托運的物品

- 🧳 行李類（托運/大件）
衣物類、盥洗用品、美妝保養、藥品、3C選帶、雜項、行李箱

適合放在托運行李箱的物品

## 7. 視覺化標籤設計
所有物品以「一排過標籤式」顯示，美觀清晰：

藍色數字：表示物品數量

✓標記：表示「建議攜帶」但不顯示數量（如證件、衛生棉）

灰色「選帶」：表示可依個人需求決定是否攜帶

ml標示：瓶罐類物品顯示建議容量
